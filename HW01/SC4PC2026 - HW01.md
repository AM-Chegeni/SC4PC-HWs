## 3. Configure Local Ubuntu Machine for SSH Connection

### 4.1 Prepare SSH Key on Local Machine

### Move and Secure the Private Key

```bash
# Create .ssh directory if it doesn't exist
mkdir -p ~/.ssh

# Set correct permissions (required for SSH)
chmod 600 ~/.ssh/**SC4PC-keys**
```

### 4.2 Create SSH Config File

### Edit SSH Config

```bash
# Open or create SSH config file
nano ~/.ssh/config

```

### Add Instance Configuration

Information for the **`Host cloudveneto-gate`  is in the email you recieve from cloudveneto manager. You should recieve your gate, user name and initial password (that should be change during the first login).**

**Information** `chegenia` is your instance settings 

```bash
Host cloudveneto-gate
    HostName gate.cloudveneto.it
    User achegen

Host chegenia # instance name
    HostName 10.67.20.144 # **IP Address**
    User almalinux # based on your VM machine you select in the Source tab 
    IdentityFile ~/.ssh/SC4PC-keys # prive key pair you copy from instance key pair making
    ProxyJump cloudveneto-gate
```

### Set Config File Permissions

```bash
chmod 600 ~/.ssh/config

```

### Connect Using Alias

```bash
# Now you can simply use:
ssh chegenia
```
